------EXTRUCTURE of a React project in VScode adn How to run it--------

---1. Project Structure---
my-react-app/
├── public/
│   └── index.html          # The HTML shell for your app. The static HTML file where React injects your app.
│   └── favicon.ico         # Optional: browser tab icon
│   └── manifest.json       # Optional: PWA support
│   └── assets/             # Optional: static images, fonts
│
├── src/ 
│   ├── components/         # Reusable UI components (Header, Footer, etc.). Break your UI into small, reusable pieces.
│   ├── pages/              # Page-level components (Home, About, etc.). Organize full views or screens (like Home, Login, Dashboard).
│   ├── assets/             # Images, icons, fonts used in JSX. Store images, logos, and fonts used in JSX.
│   ├── styles/             # CSS or SCSS files. Keep your CSS organized and modular.
│   ├── App.js              # Main app component. Your main component — often holds routing and layout.
│   ├── index.js            # Entry point for ReactDOM.render. The entry point that renders your root component (<App />) into the DOM.
│   └── setupTests.js       # Optional: for testing setup
│
├── .gitignore              # Files to ignore in Git
├── package.json            # Project metadata and dependencies
├── README.md               # Project overview
└── node_modules/           # Installed packages (auto-generated)

---2. Install Dependencies---
*In the project folder, run in the terminal:

    =>nstall Dependencies
        *imp Install

---3. Set Environment Variable (Node v17+ Fix)---
*Run in the terminal:

=>export NODE_OPTIONS=--openssl-legacy-provider

---4. Start the Development ServerStart---
*Run in the terminal:

=>imp start
                    This will:
                    *Compile your React code

                    *Launch a local development server

                    *Automatically open your browser at http://localhost:3000
